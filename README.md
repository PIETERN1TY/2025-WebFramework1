# No.8

한성대학교 25-2 웹프레임워크1 N

8조 블루 레모네이드

<br>

No.8은 대인화된 대시보드를 구성하는 Front-end only Application입니다.

위젯 형식의 대시보드를 지원하며 기본 제공 위젯 이외에 iframe embed 등 사이트 자체에서 지원하는 위젯도 적용 가능합니다.


## 주요 기능

- 드래그 & 리사이즈 그리드: 위젯의 크기와 위치를 자유롭게 조정 가능합니다.
- 다양한 위젯 제공: API 기반 위젯과 iframe embed 위젯 등을 추가 가능합니다.
- 레이아웃 저장: Firebase 연동으로 직접 설정한 레이아웃을 저장 가능합니다.

## 실행 방법

1. **의존성 설치**: 프로젝트에 필요한 라이브러리들을 설치합니다. 프로젝트의 루트 디렉터리에서 다음 명령어를 실행하세요.

   ```sh
   npm install
   ```

2. **개발 서버 실행**: 의존성 설치가 완료되면, 개발 서버를 시작합니다. 다음 명령어를 실행하세요.

   ```sh
   npm start
   ```
